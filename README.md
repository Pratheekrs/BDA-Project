# Disaster Response System using Social Media Data

## Overview

The Disaster Response System using Social Media Data is a Big Data Analytics project designed to process and analyze disaster-related social media information in real time. The system integrates multiple Big Data technologies including Hadoop, Spark, Hive, MongoDB, and graph analytics tools to build a scalable disaster monitoring and response pipeline.

The project simulates real-time disaster tweet streaming, performs distributed processing and storage, generates alerts, and visualizes disaster-location relationships using analytical graphs and charts.

---

# Features

- Real-time disaster tweet stream simulation
- Distributed storage using Hadoop HDFS
- Disaster keyword filtering and processing
- Spark Streaming based analytics
- Hive / Spark SQL analytical querying
- MongoDB based disaster alert storage
- Link analytics and graph visualization
- Multiple analytical charts and visualizations
